# cryptomarket
CryptoMarket is price-tracking website for cryptoassets in the rapidly growing cryptocurrency space.

## Build Setup

requirement: Node.js 16.x.x
```bash
# install dependencies
$ npm install

# build for production and launch server
$ npm run build
$ npm run start
```
